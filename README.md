A simple ruby progrom which display our internet ip address from linux. 



### Setup ###

+ sudo apt install ruby

### Terminal Command ###

+ git clone https://github.com/r3k4t/rubyshowmyip.git
+ cd rubyshowmyip
+ sudo ruby rubyshowmyip .rb
